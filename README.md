# FoodBot Former

A food-related bot project.

## Description

This repository contains the code for FoodBot, a bot designed to assist with food ordering, recommendations, or related tasks.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Yacine-ai-tech/foodbot_former.git
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the bot:
```
python bot.py
```
